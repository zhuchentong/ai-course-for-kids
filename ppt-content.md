# 《给孩子的AI课》课程大纲

> 适用对象：小学三年级（8-9岁）

## PPT设计风格规范

> **重要规则：每个PPT页面的"讲师备注"仅供讲师教学参考，绝不能生成到PPT页面内容中。PPT页面只展示"标题"和"内容"部分。**

### 一、整体风格定位
- **风格**：可爱卡通插画风，适合8-9岁儿童
- **氛围**：温暖、活泼、有亲和力，像一本精美的儿童绘本
- **原则**：每页文字少、图片大、留白多，信息密度低
- **氛围关键词**：绘本感、手绘感、童趣、安全感、好奇心

**参考风格**：
- **插画参考**：类似"小豆子学院"风格的扁平插画风，圆润造型、饱和色彩、简洁线条
- **整体参考**：类似于"凯叔讲故事"App的儿童课程界面，温暖友好
- **配色参考**：类似于乐高教育（LEGO Education）的课件风格，明亮但不刺眼

### 二、配色方案

**品牌色板**：
- 天蓝色 #5B9BD5（科技感、清爽）— 第一章主色
- 草绿色 #6BCB77（自然、成长）— 第二章主色
- 浅紫色 #B088F9（想象力、神秘感）— 第三章主色
- 暖橙色 #FF8C42（活力、温暖）— 第四章主色
- 柠檬黄 #FFD93D（阳光、快乐）— 吉祥物天线色 + 点缀色

**章节主色分配**：见第七节

**功能色（跨章节通用）**：
- 成功/正确：草绿色 #6BCB77
- 错误/否定：柔和红 #FF6B6B
- 注意/强调：柠檬黄 #FFD93D

**背景色**：
- 浅米色 #FFF8F0（主背景，护眼温暖）
- 纯白 #FFFFFF（内容卡片背景）

**文字色**：
- 深灰 #2D3436（正文）
- 中灰 #636E72（次要文字）
- 强调色用对应页面的主色

**使用规则**：
- 每页不超过3种品牌色（背景色不算入名额）
- 固定元素豁免：吉祥物（天蓝+柠檬黄+深灰）视为跨页通用元素，不占名额
- 功能色（[TAG:通过-绿色]绿色/[TAG:否定-红色]红色）仅在侦探判断页使用，算作辅助色
- 章节间用不同主色区分章节归属感

### 三、字体规范

#### 字号层级体系

| 层级 | 字号 | 用途 |
|------|------|------|
| 封面标题 | **48pt** | 仅封面页标题（页1） |
| 章节编号 | **72pt** | 仅章节过渡页大编号（页2、16、24、30） |
| 页面标题 | **40pt** | 所有内页标题（统一40pt，不做变动） |
| 结论强调 | **32pt加粗** | 侦探判断页结论框、数据震撼页核心数字 |
| 核心概念 | **28pt** | 金句框、类比举例核心句、总结语 |
| 正文 | **24pt** | 卡片内容、知识点说明、引导语 |
| 辅助文字 | **20pt** | 流程图气泡、时间轴说明、英文释义、卡片内小字 |
| 装饰大字母 | **64pt** | 页4的A/I装饰字母 |
| 数据大数字 | **96pt** | 页26的"10倍"等视觉冲击数字 |
| 按钮文字 | **18pt加粗** | 互动投票按钮 |
| 备注 | **18pt** | 讲师备注、角落小提示 |

#### 字号使用规则
- **标题 ≥ 40pt，正文 ≥ 24pt，备注 ≥ 18pt**（最低底线）
- **例外**：章节过渡页（模板9）为全屏渐变特殊模板，编号72pt，标题40pt
- **每页文字不超过50个字**，关键词加粗或用颜色强调
- **计数规则**：1个汉字=1字，1个英文单词=1字，连续数字（如"1946"）=1字，emoji/图标不计入；金句框内容计入，讲师备注不计入，流程图/气泡中的词语不计入
- **字号层级原则**：同一页面内字号差异 ≥ 4pt，确保层级清晰；相邻层级建议差异 8-12pt

### 四、插画与图标风格
- **插画风格**：圆润扁平插画风，线条柔和，无锐角
- **课程吉祥物**：
  - 名称：小AI（可由讲师自定义）
  - 形象：一个圆脑袋的小机器人，身高约2个头长，身体圆润无棱角
  - 头部：大眼睛（占脸1/3），头顶一根小天线（发光圆球），嘴巴是弧线
  - 颜色：主体天蓝色（第一章主色），天线发光球为柠檬黄（点缀色），眼睛为深灰（正文色）
  - 表情变化（贯穿全PPT）：
    - 好奇（歪头+问号眼）→ 用于提问页
    - 惊讶（张大嘴+星星眼）→ 用于揭晓答案页
    - 开心（弯弯眼+举手）→ 用于互动游戏页
    - 思考（摸下巴+头顶齿轮）→ 用于知识讲解页
    - 温暖（微笑+抱胸）→ 用于情感类页面（第四章）
  - 出现频率：每页必现，以角落小图标或页面内互动角色两种形式出现
- **图标风格**：圆润扁平风格（色块填充，与插画统一），不用线条描边
- **表情元素**：使用自绘卡通表情（与吉祥物同一套风格），不使用系统原生emoji；内容文案中可用emoji作为装饰点缀，但UI/视觉层面一律使用自绘
- **图片原则**：优先用插画而非真实照片

### 五、布局原则
- **左图右文** 或 **上图下文**，避免纯文字页
- **大图少字**：
  - 图文主导型（封面、类比、举例页）：图片/插画 ≥ 60%
  - 知识讲解型（双卡片、图标式、侦探页）：图片/插画 ≥ 40%
  - 互动/流程型（游戏、流程图页）：图片/插画 ≥ 30%
- **卡片式排版**：内容用圆角卡片承载，卡片间有呼吸感
- **对齐方式**：
  - 页面视觉重心：主要内容区居中分布
  - 标题文字：居中对齐
  - 正文/卡片内文字：左对齐（提升可读性）
- **留白**：页边距 ≥ 15%，内容不拥挤

### 六、基础规格

**画布规格**：
- 比例：16:9（宽屏）
- 分辨率：1920×1080px
- 安全区域：距边缘5%内不放置关键内容

**吉祥物尺寸规范**：
- 封面/重点页：占页面高度30%-40%
- 互动角色（页面内参与互动）：占页面高度15%-20%
- 角落小图标：64×64px

**间距系统（基准8px）**：
- 紧凑间距：8px（卡片内元素间距）
- 标准间距：16px（卡片间距、网格间距）
- 宽松间距：24px（区域间距）
- 页面边距：≥ 15%
